
react应用程序

##### 安装package  
```
npm init -y
npm install --save-dev webpack webpack-dev-server html-webpack-plugin clean-webpack-plugin  style-loader css-loader less-loader babel-loader babel-core babel-preset-es2015  babel-preset-react babel-preset-stage-0
npm install --save react react-dom
```

##### 创建目录结构
```bat
mkdir src
mkdir build
type nul>readme.md
type nul>webpack.config.js
type nul>index.html
cd src
type nul>index.jsx
cd ..
```
